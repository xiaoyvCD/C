# ✈️ 无人机心跳监控系统（地面站版）

实时模拟无人机心跳信号，通过地面站检测连接状态，3秒未收到心跳即判定为超时。

## 🚀 功能特性

- **心跳模拟**：每秒自动生成带序号和时间戳的心跳包
- **掉线检测**：3秒未收到心跳自动标记超时（状态值=2）
- **状态可视化**：
  - 0 = 初始状态
  - 1 = 信号接收成功
  - 2 = 连接超时
- **地面站控制**：可模拟地面站在线/离线状态
- **实时监控**：动态更新折线图和事件日志
